# Chrono Track — 仕様書

## 概要

訪問したWebページとその滞在時間をローカルに記録し、ポップアップで確認・エクスポートできるFirefox拡張機能。

---

## トラッキング仕様

### カウント条件（すべて満たす場合にカウント）

- そのタブがアクティブタブである
- Firefoxウィンドウがフォーカスされている
- 記録が一時停止中でない

### カウント停止条件（いずれかを満たした場合に停止）

- 別タブに切り替えた
- 別ウィンドウまたは別アプリに切り替えた
- タブを閉じた
- 記録を一時停止した

### 記録のルール

- **最低記録時間**: 5秒未満の滞在は記録しない（ノイズ除去）
- **URLキー**: クエリパラメータを含む完全URL（例: `https://example.com/search?q=foo`）
- **集計単位**: 日付（`YYYY-MM-DD`）× URL
- **タイトル**: `tabs.onUpdated` の `changeInfo.title` を監視し、JavaScript による遅延更新にも対応

---

## データ設計

### ストレージ

- `browser.storage.local` を使用
- データは永続保持（自動削除なし）

### データ構造（訪問記録）

```json
{
  "2026-03-21": {
    "https://github.com/foo/bar?tab=readme": {
      "title": "foo/bar - GitHub",
      "visits": 3,
      "totalMs": 125000
    }
  }
}
```

| フィールド | 型 | 説明 |
|---|---|---|
| `title` | string | ページタイトル（最後に記録されたもの） |
| `visits` | number | その日の訪問回数 |
| `totalMs` | number | その日の合計滞在時間（ミリ秒） |

### データ構造（設定）

| キー | 型 | 説明 |
|---|---|---|
| `_isPaused` | boolean | 記録の一時停止状態。ブラウザ再起動後も維持される |

## UI仕様

### ポップアップ画面

- ヘッダー: ロゴ・日付・一時停止トグル
- サマリー: 今日の合計時間・訪問ページ数
- 一覧: 今日の訪問ページを**滞在時間降順**で表示
  - 各行: ページタイトル・URL・滞在時間・訪問回数

### 一時停止トグル

- ヘッダー右端にスライドトグルボタン
- **記録中**: トグルが青（右側）
- **一時停止中**: トグルがグレー（左側）＋黄色バナー「⏸ 記録を一時停止中」＋アドオンアイコンにバッジ（⏸）
- 一時停止時: その時点のセッションを安全に保存してから停止
- 再開時: アクティブタブのセッションを即座に開始

### エクスポート

- **CSV**: 今日のデータを1行1URLでエクスポート（BOM付きUTF-8）
- **JSON**: ストレージの全データをそのままエクスポート


## 必要なFirefox API / 権限

| 権限 | 用途 |
|---|---|
| `tabs` | アクティブタブのURL・タイトル取得、タブ切り替え検知 |
| `storage` | ローカルデータの読み書き（訪問記録・一時停止状態） |
| `windows` API | ウィンドウのフォーカス状態検知（権限宣言不要） |
