# Chrono Track — 仕様書

## 概要

訪問したWebページとその滞在時間をローカルに記録し、ポップアップで確認・エクスポートできるFirefox拡張機能。

---

## トラッキング仕様

### カウント条件（すべて満たす場合にカウント）

- そのタブがアクティブタブである
- Firefoxウィンドウがフォーカスされている
- 記録が一時停止中でない

### カウント停止条件（いずれかを満たした場合に停止）

- 別タブに切り替えた
- 別ウィンドウまたは別アプリに切り替えた
- タブを閉じた
- 記録を一時停止した

### 記録のルール

- **最低記録時間**: 5秒未満の滞在は記録しない（ノイズ除去）
- **URLキー**: クエリパラメータを含む完全URL（例: `https://example.com/search?q=foo`）
- **集計単位**: 日付（`YYYY-MM-DD`）× URL
- **タイトル**: `tabs.onUpdated` の `changeInfo.title` を監視し、JavaScript による遅延更新にも対応

---

## データ設計

### ストレージ

- `browser.storage.local` を使用
- データは永続保持（自動削除なし）

### データ構造（訪問記録）

```json
{
  "2026-03-21": {
    "https://github.com/foo/bar?tab=readme": {
      "title": "foo/bar - GitHub",
      "visits": 3,
      "totalMs": 125000
    }
  }
}
```

| フィールド | 型 | 説明 |
|---|---|---|
| `title` | string | ページタイトル（最後に記録されたもの） |
| `visits` | number | その日の訪問回数 |
| `totalMs` | number | その日の合計滞在時間（ミリ秒） |

### データ構造（設定）

| キー | 型 | 説明 |
|---|---|---|
| `_isPaused` | boolean | 記録の一時停止状態。ブラウザ再起動後も維持される |

## UI仕様

### ポップアップ画面（今日ビュー）

- ヘッダー: ロゴ・**今日の**日付・一時停止トグル
- サマリー: **今日の**合計時間・訪問ページ数
- 一覧: **今日の**訪問ページを滞在時間降順で表示
  - 各行: ページタイトル・URL・滞在時間・訪問回数
- 日付が変わると自動的に0からカウントし直す（過去のデータは表示しない）
- 履歴ページへのリンクボタン

### 履歴ページ（history.html）

- 記録がある日付を新しい順に一覧表示
- 日付をクリックするとその日の詳細（ページ一覧・滞在時間）を展開表示
- 各日のサマリー（合計時間・訪問ページ数）を表示

#### 履歴ページのエクスポート

- 各日付にチェックボックスを表示
- 「全選択 / 全解除」ボタンでまとめて選択できる
- 選択した日付のデータを **CSV** または **JSON** でエクスポート
- CSV: 選択した全日付のデータを1行1URLで出力（日付カラムを含む）
- JSON: 選択した日付のキーのみを含むオブジェクトで出力

### 一時停止トグル

- ヘッダー右端にスライドトグルボタン
- **記録中**: トグルが青（右側）
- **一時停止中**: トグルがグレー（左側）＋黄色バナー「⏸ 記録を一時停止中」＋アドオンアイコンにバッジ（⏸）
- 一時停止時: その時点のセッションを安全に保存してから停止
- 再開時: アクティブタブのセッションを即座に開始

### エクスポート

- **CSV / JSON ともに今日のデータのみ**をエクスポート（BOM付きUTF-8）

## ファイル構成

```
chrono-track/
├── manifest.json          # 拡張機能のメタデータと権限設定
├── background/
│   └── tracker.js         # トラッキングのメインロジック（バックグラウンドスクリプト）
├── popup/
│   ├── popup.html         # ポップアップのHTML（今日ビュー）
│   ├── popup.css          # ポップアップ・履歴ページ共通スタイル
│   ├── popup.js           # ポップアップのロジック
│   ├── history.html       # 履歴ページのHTML
│   └── history.js         # 履歴ページのロジック
└── icons/
    ├── icon.svg           # 拡張機能アイコン（ソース）
    ├── icon-48.png        # アイコン 48px
    └── icon-96.png        # アイコン 96px
```

## 必要なFirefox API / 権限

| 権限 | 用途 |
|---|---|
| `tabs` | アクティブタブのURL・タイトル取得、タブ切り替え検知 |
| `storage` | ローカルデータの読み書き（訪問記録・一時停止状態） |
| `windows` API | ウィンドウのフォーカス状態検知（権限宣言不要） |
