# Chrono Track — 仕様書

## 概要

訪問したWebページとその滞在時間をローカルに記録し、ポップアップで確認・エクスポートできるFirefox拡張機能。

---

## トラッキング仕様

### カウント条件（すべて満たす場合にカウント）

- そのタブがアクティブタブである
- Firefoxウィンドウがフォーカスされている

### カウント停止条件（いずれかを満たした場合に停止）

- 別タブに切り替えた
- 別ウィンドウまたは別アプリに切り替えた
- タブを閉じた

### 記録のルール

- **最低記録時間**: 5秒未満の滞在は記録しない（ノイズ除去）
- **URLキー**: クエリパラメータを含む完全URL（例: `https://example.com/search?q=foo`）
- **集計単位**: 日付（`YYYY-MM-DD`）× URL

---

## データ設計

### ストレージ

- `browser.storage.local` を使用
- データは永続保持（自動削除なし）

### データ構造

```json
{
  "2026-03-21": {
    "https://github.com/foo/bar?tab=readme": {
      "title": "foo/bar - GitHub",
      "visits": 3,
      "totalMs": 125000
    }
  }
}
```

| フィールド | 型 | 説明 |
|---|---|---|
| `title` | string | ページタイトル（最後に記録されたもの） |
| `visits` | number | その日の訪問回数 |
| `totalMs` | number | その日の合計滞在時間（ミリ秒） |

---

## UI仕様

### ポップアップ画面

- 今日の訪問ページ一覧を**滞在時間降順**で表示
- 各行に表示する情報: ページタイトル・URL・滞在時間・訪問回数
- ヘッダーに今日の合計時間・合計訪問数を表示

### エクスポート

- **CSV**: 今日のデータを1行1URLでエクスポート
- **JSON**: ストレージの全データをそのままエクスポート

---

## 必要なFirefox API / 権限

| 権限 | 用途 |
|---|---|
| `tabs` | アクティブタブのURL・タイトル取得 |
| `storage` | ローカルデータの読み書き |
| `windows` | ウィンドウのフォーカス状態検知 |
